# Full Stack E-Commerce Application

## Overview
A full-stack e-commerce web application developed using React.js, FastAPI, and PostgreSQL.

## Features
- User Authentication (JWT)
- Product Management
- Shopping Cart
- Order Management
- Admin Panel
- REST API Development
- CRUD Operations

## Tech Stack
- Frontend: React.js
- Backend: FastAPI
- Database: PostgreSQL
- Version Control: Git & GitHub

## Project Structure

```
├── frontend/
├── product/
└── userManageApi/
```
