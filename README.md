# 🛒 Full Stack E-Commerce Application

## 📖 Overview
A Full Stack E-Commerce Application developed using React.js, FastAPI, and PostgreSQL (Supabase). The application provides secure user authentication, product management, shopping cart, order management, and admin functionality.

---

## ✨ Features

- User Registration & Login
- JWT Authentication
- Product Management
- Shopping Cart
- Order Management
- Admin Dashboard
- RESTful APIs
- CRUD Operations

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- FastAPI
- Python

### Database
- PostgreSQL (Supabase)

### Deployment
- Netlify (Frontend)
- Render (Backend)
